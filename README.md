# Game of life
Simple implementation of [Conway's Game of Life](https://en.wikipedia.org/wiki/Conway%27s_Game_of_Life)

# How to launch
## Prerequisites
- `g++` compiler
- `make`

With default settings:
```console
$ make
```
With custom height and width
```console
$ make build
$ ./gof --h [HEIGHT] --w [WIDTH]
```
